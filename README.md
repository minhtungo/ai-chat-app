- **Frontend Framework**: React
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS
- **State Management**:

- **Data Fetching**:

- **Schema Validation**:
  - Zod
- **UI Components**:
  - Radix UI primitives
  - Shadcn/ui components
- **Development Tools**:
  - TypeScript
  - Vite
  - ESLint
